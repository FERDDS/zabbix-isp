

# Prototipo de Sistema de Monitorización

## Descripción
Sistema de monitoreo centralizado y automatizado para infraestructura de ISP implementado con Zabbix 6.4, utilizando el protocolo SNMP v2c para la recolección de métricas de dispositivos de red Cisco CSR1000v.

## Tecnologías utilizadas
- **Zabbix 6.4** — Plataforma de monitoreo
- **MariaDB 10.11** — Base de datos
- **Apache2** — Servidor web
- **Debian 12** — Sistema operativo del servidor
- **EVE-NG** — Simulador de red
- **Cisco CSR1000v IOS-XE 16.12.1a** — Routers simulados
- **SNMP v2c** — Protocolo de monitoreo
- **OSPF** — Protocolo de enrutamiento
